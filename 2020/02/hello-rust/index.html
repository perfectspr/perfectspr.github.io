Hello Rust

  2020-02-09   编程 rust   rust 0  评论 字数统计: 513字 阅读时长: 2分

最近收到许多Rust相关推荐文章，看起来很诱人，准备开始学习Rust。

Mac开发环境搭建

安装Rust工具包Restup:

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

安装完成后，将cargo可执行目录放到Path环境变量，然后检查cargo的版本。

$ echo 'export PATH=$HOME/.cargo/bin:$PATH' >> ~/.bash_profile
$ . ~/.bash_profile
$ cargo --version

cargo是Rust的构建工具，类似npm和maven. cargo的包叫做crate(箱子). https://crates.io/. 没有VPN的话，建议用清华的源:

$ echo 'export RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup' >> ~/.bash_profile
$ . ~/.bash_profile

安装Vim插件coc-rls. Coc.nvim目前是我最喜欢的Vim自动完成插件, 通过Language-Server它可以支持大部分编程语言。语法高亮插件用vim-polyglot，它支持绝大部分语言的语法高亮。

:CocInstall coc-rls

Hello World

# 创建项目
$ cargo new hello-rust
$ cd hello-rust
$ cargo run

添加依赖

打开Rust的项目配置文件Cargo.toml,加入以下依赖

ferris-says = "1.1"

然后安装依赖:

$ cargo build

类似npm，此命令会在当前目录下生成Cargo.lock文件。Rust借鉴了现代语言的很多最佳实践,舒服:)

写个小程序

打开src/main.rs:

extern crate ferris_says;

use ferris_says::say;
use std::io::{ stdout, BufWriter };

fn main() {
    let stdout = stdout();
    let out = b"Hello fellow Rustaceans!";
    let width = 24;

    let mut writer = BufWriter::new(stdout.lock());
    say(out, width, &mut writer).unwrap();
}

然后运行

$ cargo run

输出以下内容：
